# TPRM AI Reviewer

An AI-assisted Third Party Risk Management (TPRM) system that analyzes vendor documents (MSA and SoW) and automatically answers compliance questionnaires using Agentic Retrieval-Augmented Generation (RAG).

The system provides **evidence-backed answers, human review capability, and workflow tracking** to support enterprise vendor onboarding.

---

# Problem

Vendor onboarding teams must manually review large legal documents such as:

- Master Service Agreements (MSA)
- Statements of Work (SoW)

From these documents, they must answer risk questionnaires related to:

- Data security
- Data privacy
- Architecture
- Compliance

This process is slow, manual, and difficult to scale.

---

# Solution

TPRM AI Reviewer automates document analysis and produces **traceable answers with evidence and rationale**.

Key capabilities:

- Upload vendor documents
- Extract and chunk document content
- Retrieve relevant evidence
- Generate answers using an LLM
- Allow human reviewers to validate or override
- Track workflow status
- Provide operational dashboard
